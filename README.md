本示范项目是基于Angular2和Rxjs技术构建的小型流式应用，以真实的瑞典宜家(IKEA)家具产品的购物场景为依托，涵盖的业务功能包括商品展示、购物车、已购商品列表、基于给定条件的商品筛选，等等。
